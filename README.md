# 📈 Linear Regression Project

A Machine Learning project that predicts continuous values using the **Linear Regression** algorithm. This project demonstrates the complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), model training, model evaluation, and prediction using Python.

---

## 📌 Project Overview

This project uses a Linear Regression model to analyze an insurance dataset and predict charges based on different input features.

The project covers:
- Data Loading
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Selection
- Train-Test Split
- Linear Regression Model
- Model Evaluation
- Prediction

---

## 🛠️ Technologies Used

- Jupyter Notebook
